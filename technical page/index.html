Introduction

Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.js and has a rich ecosystem of extensions for other languages and runtimes.

VS Code combines the simplicity of a code editor with what developers need for their core edit-build-debug cycle. It provides comprehensive code editing, navigation, and understanding support along with lightweight debugging, a rich extensibility model, and lightweight integration with existing tools.

Key features of VS Code include intelligent code completion, debugging support, built-in Git integration, and extensive customization options. It's designed to be fast and responsive while providing powerful development tools.

  • IntelliSense - Smart code completions based on variable types, function definitions, and imported modules
  • Debugging - Debug code right from the editor. Launch or attach to your running apps and debug with break points, call stacks, and an interactive console
  • Built-in Git - Working with Git and other SCM providers has never been easier. Review diffs, stage files, and make commits right from the editor
  • Extensions - Install extensions to add new languages, themes, debuggers, and to connect to additional services
  • Customization - Customize every aspect of your editor including themes, keybindings, and settings

VS Code is free and open source, developed by Microsoft and the community. It receives monthly updates with new features and bug fixes, ensuring you always have access to the latest tools and improvements.

code --version
Getting Started

Getting up and running with Visual Studio Code is quick and easy. It is a small download so you can install it in a matter of minutes and give VS Code a try.

To get started with VS Code, you'll want to download and install it from the official website. VS Code is updated monthly with new features and bug fixes. You can download it for Windows, macOS, and Linux on the Visual Studio Code download page.

After installation, you can immediately start using VS Code. Opening a project folder is as simple as using the File menu or command line.

code /path/to/your/project

You can also open VS Code from the command line by typing the following in your project directory. This will open the current directory in VS Code and load all your project files.

code .

User Interface

At its heart, Visual Studio Code is a code editor. Like many other code editors, VS Code adopts a common user interface and layout of an explorer on the left, showing all of the files and folders you have access to, and an editor on the right, showing the content of the files you have opened.

The VS Code user interface is divided into five main areas that work together to provide a seamless development experience.

  • Editor - The main area to edit your files. You can open as many editors as you like side by side vertically and horizontally
  • Side Bar - Contains different views like the Explorer to assist you while working on your project
  • Status Bar - Information about the opened project and the files you edit
  • Activity Bar - Located on the far left-hand side, lets you switch between views
  • Panels - You can display different panels below the editor region for output or debug information, errors and warnings, or an integrated terminal

Each time you start VS Code, it opens up in the same state it was in when you last closed it. The folder, layout, and opened files are preserved.

You can access the Command Palette to execute commands with the following shortcut:

Ctrl+Shift+P F1
Basic Editing

Visual Studio Code is a powerful editor that's easy to use for basic editing tasks. VS Code has great support for many different programming languages and has IntelliSense code completion, snippets, and much more.

Here are some basic editing features you should know about to improve your productivity:

  • Multiple selections (multi-cursor) - Add additional cursors to select and edit text in multiple locations
  • Column (box) selection - Select blocks of text using column selection mode
  • Fast scrolling - Press the Alt key to fast scroll through your file
  • Copy line up/down - Quickly duplicate lines of code
  • Move line up/down - Reorder lines of code efficiently
  • Find and Replace - Search and replace text with powerful pattern matching

To add multiple cursors, you can use the following keyboard shortcuts:

Alt+Click Ctrl+Alt+Down Ctrl+Alt+Up

VS Code also has rich formatting support. You can format your entire document or just the current selection with the Format Document and Format Selection commands.

Shift+Alt+F
Extensions

Extensions let you add languages, debuggers, and tools to your installation to support your development workflow. VS Code's rich extensibility model lets extension authors plug directly into the VS Code UI and contribute functionality through the same APIs used by VS Code.

You can browse and install extensions from within VS Code. Bring up the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of VS Code or using the View: Extensions command.

Ctrl+Shift+X

Popular extensions include tools for languages, themes, formatters, and productivity enhancements.

  • ESLint - Integrates ESLint JavaScript into VS Code
  • Prettier - Code formatter using prettier
  • Python - Linting, debugging, IntelliSense, and more for Python
  • GitLens - Supercharge Git within VS Code
  • Live Server - Launch a local development server with live reload
  • Bracket Pair Colorizer - Color matching brackets for better code readability

To install an extension, simply click the Install button. Once the installation is complete, the Install button will change to a Manage gear button.

You can search for extensions using keywords and filter by category, popularity, or rating. Extensions can dramatically enhance your development experience.

ext install publisher.extension-name
Settings

VS Code provides different scopes for settings. When you open a workspace, you will see at least the following two scopes: User Settings and Workspace Settings. User settings apply globally to any instance of VS Code you open, while Workspace settings apply specifically to the current workspace.

To modify user settings, you can open the settings editor using the following keyboard shortcut:

Ctrl+,

You can also edit the settings.json file directly. Here's an example configuration:

{ "editor.fontSize": 14, "editor.tabSize": 2, "files.autoSave": "afterDelay", "workbench.colorTheme": "Dark+" }

Common settings to configure include font preferences, editor behavior, and theme selection.

  • Font size and family
  • Tab size and indentation
  • Auto save behavior
  • Color theme
  • File associations
  • Line numbers and rulers

Settings are stored in JSON format and can be easily shared across machines or with team members. You can also create workspace-specific settings that override user settings.

Keyboard Shortcuts

Visual Studio Code has a rich set of default keyboard shortcuts as well as allowing you to customize them. You can view all default keyboard shortcuts by opening the Keyboard Shortcuts editor.

Ctrl+K Ctrl+S

Here are some essential keyboard shortcuts to increase your productivity:

  • Ctrl+P - Quick Open, Go to File
  • Ctrl+Shift+N - New window/instance
  • Ctrl+Shift+W - Close window/instance
  • Ctrl+X - Cut line (empty selection)
  • Ctrl+C - Copy line (empty selection)
  • Ctrl+V - Paste
  • Ctrl+Z - Undo

You can also search for shortcuts and add your own keybindings in the keybindings.json file:

[ { "key": "ctrl+shift+alt+c", "command": "editor.action.commentLine" } ]

Custom keyboard shortcuts can help you work more efficiently by tailoring the editor to your workflow. You can bind any command to any key combination you prefer.
